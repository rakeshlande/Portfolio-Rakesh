We learned Autodesk fusion 360 from basic :-


1. Sketch Tools:

- Line: Create straight or curved lines.
- Rectangle: Draw rectangles or squares.
- Circle: Draw circles or arcs.
- Arc: Create arcs of specified radius.
- Polygon: Draw regular polygons with a defined number of sides.
- Fillet: Round off sharp corners of sketch entities.
- Trim: Remove parts of sketch entities that overlap.
- Dimension: Add dimensions to sketch geometry.

2. Modeling Tools:

- Extrude: Create 3D features by extending 2D sketches along a specified direction.
- Revolve: Generate 3D features by revolving 2D sketches around an axis.
- Sweep: Create complex shapes by sweeping a profile along a path.
- Loft: Blend between two or more profiles to create complex shapes.
- Fillet/Chamfer: Round off or bevel edges of 3D models.
- Hole: Add standard or custom holes to models.

3. Assembly Tools:

- Joint: Connect components together with different types of joints such as revolute, slider, or cylindrical.
- As-Built Joint: Create ad-hoc connections between components.
- Motion Study: Simulate the motion of assembled components.
- Contact Sets: Define interactions between components.

4. Modify Tools:

- Move/Copy: Translate and duplicate objects.
- Pattern: Create linear, circular, or rectangular patterns of objects.
- Mirror: Mirror geometry across a plane or face.
- Shell: Remove material from the interior of a solid to create a hollow part.
- Combine: Merge or subtract multiple bodies to create complex shapes.
